# 变更摘要

## V0.02 — 音视频播放器功能完成

### 新增功能

- **音频播放**：基于 FFmpeg 解码 + QAudioSink 输出，支持播放/暂停/停止/Seek
- **音频设备切换**：ComboBox 下拉选择音频输出设备，实时切换
- **音量控制**：自定义滑块，支持静音切换
- **倍速播放**：0.5x ~ 2.0x 可调
- **双缓冲视频渲染**：QML 双 Image 交替显示，消除画面闪烁
- **文件日志系统**：`qInstallMessageHandler` 输出日志到 `audiolog.txt`

### 关键问题修复

1. **QAudioSink 无声**：AudioPullDevice::bytesAvailable() 必须返回正值才能触发 readData() 回调
2. **音频队列预填**：启动 QAudioSink 前预解码 ~50ms PCM 数据，防止立即 underrun
3. **FFmpeg 缓冲区管理**：从 av_malloc + av_image_fill_arrays 改为 av_frame_get_buffer()，消除 double-free 风险
4. **日志处理器退出清理**：app.exec() 返回后恢复默认 handler，防止静态析构时访问已销毁对象
5. **格式协商**：QAudioSink 实际格式与设置格式不一致时自动重配 SwrContext
6. **Debug/Release CRT 兼容**：vcpkg 只有 Release 库时，Debug 构建的 _CtrlsValidHeapPointer 断言为假阳性
