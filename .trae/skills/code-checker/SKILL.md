# 代码检查 Agent

## 功能

- 检查代码中的语法错误和编译问题
- 分析代码逻辑和架构问题
- 识别性能优化机会
- 检查代码风格和最佳实践
- 提供详细的问题报告和修复建议

## 使用方法

当用户要求检查代码或在开发过程中需要代码审查时，调用此Agent。Agent会分析项目代码，识别问题，并提供详细的报告和修复建议。

## 检查范围

- C++代码（.cpp, .h文件）
- QML代码（.qml文件）
- CMake配置文件（CMakeLists.txt）
- 项目结构和依赖关系

## 输出格式

- 问题描述：详细说明发现的问题
- 位置：文件路径和行号
- 严重程度：高、中、低
- 修复建议：具体的修复方法和代码示例
